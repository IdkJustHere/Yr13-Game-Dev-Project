Foundation — Grid, Tiles & Item Flow
Weeks 1–4  ·  Solo
By the end of this sprint, you can place conveyor tiles on a grid, press Space to start, and watch coloured cubes slide from a Source node to a Sink node. Placeholder visuals only — coloured boxes are perfectly fine.
Week 01 Project Setup & Grid Foundation

Autoload: GameManager

Create autoloads/GameManager.gd and register it as an Autoload singleton in Project Settings → Autoload. This is the central nervous system of the game — it holds global state: current phase (PLANNING / RUSH_HOUR), the tick timer, current level config, inventory counts per tile type, and delivery quota progress. See Appendix A.

🎬

Root: Node3D

The master scene. Everything lives under this node. Set it as the Main Scene in Project Settings → Application → Run.

🗂
scenes/GridManager.tscn
Root: Node3D

Manages the logical grid — a Dictionary mapping Vector2i cell coordinates to tile data — and renders floor tiles. No child nodes; tiles are spawned into it at runtime.

Attach scripts/GridManager.gd. Exposes: place_tile(cell, tile_scene, rotation), remove_tile(cell), get_tile(cell), cell_to_world(cell), world_to_cell(pos). See Appendix B.

⚠️
Godot's 3D world uses X/Z as the horizontal plane (Y is up). Grid cells map to Vector2i(x, z). World position is Vector3(cell.x, 0, cell.y). This trips up almost everyone at first.
🎮
scenes/PlacementController.tscn
Root: Node3D

Handles all mouse input for placing and removing tiles. Contains a GhostTile child that follows the cursor.

PlacementController : Node3D
GhostTile : MeshInstance3D — translucent box (cyan, alpha ~0.5) follows cursor
Raycast : RayCast3D — shoots ray from camera to mouse position

In _unhandled_input(): cast a ray from the camera through the mouse position. If it hits the floor plane (Y=0), snap the hit point to the nearest grid cell and move GhostTile there. LMB calls GridManager.place_tile(), RMB calls GridManager.remove_tile(). R rotates. 1/2/3 change tile type. See Appendix C.

⚠️
Raycasting to an infinite plane requires a Plane object and plane.intersects_ray() — not a physics body. See Appendix C for the exact pattern.

Add a temporary flat MeshInstance3D (BoxMesh, scale 10×0.1×10) under GameWorld as a floor visual. Place a StaticBody3D + CollisionShape3D beneath it so the raycast has something to hit during early testing.

scenes/ConveyorTile.tscn
Root: Node3D

A single conveyor tile. Placeholder: a dark-grey BoxMesh with a small arrow MeshInstance3D child to show direction.

ConveyorTile : Node3D
Mesh : MeshInstance3D — BoxMesh placeholder (dark grey)
ArrowMesh : MeshInstance3D — tiny elongated box pointing in tile's facing direction

⚠️
Direction logic is the trickiest part. Straight tile facing North: item enters from South → exits North. Left corner facing North: enters from South → exits West. Build a lookup table and test it manually before connecting it to item movement.

Week 04 Tick System & Item Movement Integration

Wire everything together through the GameManager tick system. In GameManager.gd, use _process(delta) to accumulate time and fire a tick signal every tick_rate seconds (default 0.5s) during Rush Hour. Each tick: check for collisions first, then emit tick. Sources and ItemCrates listen to tick via signal connections. See Appendix A.

Add Spacebar handling in GameWorld.gd: if Input.is_action_just_pressed("toggle_phase"): GameManager.toggle_phase(). Define toggle_phase in Project Settings → Input Map. See Appendix P for all required Input Map actions.

Game Systems — Jams, Scoring, UI & Levels

Week 05Jam Detection & Fail States

Jam detection runs at the top of each tick, before any items move. Logic: build a Dictionary mapping each Vector2i cell to how many items will move into it this tick. Any cell with 2+ incoming items = Jam.

In GameManager.gd, add: jam_count: int (starts at 0), MAX_JAMS = 3, jammed_cells: Dictionary. On a Jam: emit jam_occurred(cell), increment jam_count. If jam_count >= MAX_JAMS, emit kitchen_shutdown and end the run. See Appendix A.

Jam clearing: during Rush Hour, if LMB is clicked on a jammed cell, call GameManager.clear_jam(cell) — destroys one item, subtracts 10 points, removes the cell from the jammed set. See Appendix A.

SignalEmitted byConnect to
jam_occurred(cell)GameManagerHUD._on_jam_occurred
jam_occurred(cell)GameManagerGridManager._on_jam_occurred (spawn particles later)
kitchen_shutdownGameManagerGameWorld._on_kitchen_shutdown
level_timeoutGameManagerGameWorld._on_level_timeout
⚠️
Jammed items must be paused — not moved in subsequent ticks. Add is_jammed: bool to ItemCrate and check it at the start of _on_tick().
Week 06Scoring, Timer & Level Config Resource

Add to GameManager.gd: score, deliveries, quota, time_remaining, tiles_used, tiles_available, tick_rate. Load these from a LevelConfig Resource.

Scoring: on delivery → score += 100, deliveries += 1. If deliveries reach quota → level_complete(). Timer counts down during Rush Hour only. On zero → level_timeout().

Grading Logic

In level_complete():
— Bronze: quota delivered before timer expires
— Silver: quota delivered with ≥30s remaining
— Gold: Silver conditions AND tiles_used < tiles_available / 2

📄
scripts/LevelConfig.gd
extends Resource

A class_name LevelConfig Resource with @export variables: level_name, level_index, grid_size: Vector2i, quota, time_limit, tile_budget, tick_rate, obstacle_cells: Array[Vector2i], source_configs: Array, sink_configs: Array. Save one .tres file per level. See Appendix G.

⚠️
Use a custom Resource (not JSON) for level data. It lets you edit values in the Godot Inspector directly.

scenes/GameWorld.tscn — Particle FX
GPUParticles3D nodes

Add two GPUParticles3D nodes to GameWorld.tscn, both starting at (0,0,0). They are teleported to the target cell when triggered.

JamParticles — Red Spark Burst

Emitting = false, Amount = 16, Lifetime = 0.4, One Shot = true, Explosiveness = 0.9. Process Material: Direction=(0,1,0), Spread=60°, Velocity Min/Max=2/4, Scale=0.05–0.1, Color=#FF4444. Trigger: on jam_occurred(cell), move to cell's world position, set emitting = true, call restart().

DeliveryParticles — White Smoke Puff

Amount = 8, Lifetime = 0.6, One Shot = true, Explosiveness = 0.85. Color=white, Scale=0.08–0.15, Spread=80°. Triggered by item_delivered at SinkNode's position. See Appendix K.

⚠️
GPUParticles3D.restart() only works when emitting = true. Set position → set emitting → call restart. One-shot particles auto-stop after their lifetime.

# autoloads/GameManager.gd  —  register as Autoload singleton
extends Node

enum Phase { PLANNING, RUSH_HOUR }
enum ItemType { MEAT, FRUIT, SPICE }

signal tick
signal phase_changed(new_phase)
signal jam_occurred(cell: Vector2i)
signal kitchen_shutdown
signal level_timeout
signal score_changed(new_score)
signal inventory_changed

var phase: Phase = Phase.PLANNING
var score: int = 0
var deliveries: int = 0
var quota: int = 20
var time_remaining: float = 90.0
var jam_count: int = 0
const MAX_JAMS = 3
var jammed_cells: Dictionary = {}
var tile_inventory: Dictionary = {}
var tiles_used: int = 0
var tiles_available: int = 12
var tick_rate: float = 0.5
var is_sandbox: bool = false
var current_config: LevelConfig
var _all_items: Array = []
var _tick_timer: float = 0.0

func _process(delta: float) -> void:
    if phase != Phase.RUSH_HOUR: return
    time_remaining -= delta
    if time_remaining <= 0.0: level_timeout.emit(); return
    _tick_timer += delta
    if _tick_timer >= tick_rate:
        _tick_timer -= tick_rate
        _do_tick()

func _do_tick() -> void:
    # 1. Check collisions BEFORE moving
    var next_positions: Dictionary = {}
    for item in _all_items:
        if item.is_jammed: continue
        var next = item.get_next_cell()
        if next in next_positions: next_positions[next].append(item)
        else: next_positions[next] = [item]
    # 2. Flag jams
    for cell in next_positions:
        if next_positions[cell].size() > 1: _trigger_jam(cell, next_positions[cell])
    # 3. Emit tick (items move in _on_tick listeners)
    tick.emit()

func _trigger_jam(cell: Vector2i, items: Array) -> void:
    for item in items: item.is_jammed = true
    jammed_cells[cell] = items
    jam_count += 1
    jam_occurred.emit(cell)
    if jam_count >= MAX_JAMS: kitchen_shutdown.emit()

func toggle_phase() -> void:
    phase = Phase.RUSH_HOUR if phase == Phase.PLANNING else Phase.PLANNING
    phase_changed.emit(phase)

func clear_jam(cell: Vector2i) -> void:
    if not cell in jammed_cells: return
    var items = jammed_cells[cell]
    if items.size() > 0: items[0].queue_free(); items.pop_front()
    if items.size() <= 1:
        jammed_cells.erase(cell)
        if items.size() == 1: items[0].is_jammed = false
    score -= 10; score_changed.emit(score)

func _on_item_delivered() -> void:
    deliveries += 1; score += 100; score_changed.emit(score)
    if not is_sandbox and deliveries >= quota: _level_complete()

func _level_complete() -> void:
    toggle_phase()
    var grade = 1
    if time_remaining >= 30.0: grade = 2
    if time_remaining >= 30.0 and tiles_used < tiles_available / 2: grade = 3
    level_complete.emit(grade)

# scripts/GridManager.gd
extends Node3D
var grid: Dictionary = {}
var obstacles: Dictionary = {}
var grid_size: Vector2i = Vector2i(8, 8)
const TILE_SIZE: float = 1.0

func cell_to_world(cell: Vector2i) -> Vector3:
    return Vector3(cell.x * TILE_SIZE, 0.0, cell.y * TILE_SIZE)

func world_to_cell(pos: Vector3) -> Vector2i:
    return Vector2i(roundi(pos.x / TILE_SIZE), roundi(pos.z / TILE_SIZE))

func is_valid_cell(cell: Vector2i) -> bool:
    return cell.x >= 0 and cell.x < grid_size.x \
       and cell.y >= 0 and cell.y < grid_size.y \
       and not cell in obstacles

func place_tile(cell: Vector2i, tile_scene: PackedScene, rot: int) -> bool:
    if not is_valid_cell(cell) or cell in grid: return false
    var tile = tile_scene.instantiate()
    add_child(tile)
    tile.position = cell_to_world(cell)
    tile.rotation_degrees.y = rot * 90.0
    tile.facing = rot
    grid[cell] = tile; return true

func remove_tile(cell: Vector2i) -> bool:
    if not cell in grid: return false
    grid[cell].queue_free(); grid.erase(cell); return true

func get_tile(cell: Vector2i) -> Node3D: return grid.get(cell, null)

DConveyorTile.gd — Direction Logic
# Directions: 0=North(-Z)  1=East(+X)  2=South(+Z)  3=West(-X)
enum TileType { STRAIGHT, CORNER_L, CORNER_R }
@export var tile_type: TileType = TileType.STRAIGHT
var facing: int = 0

func get_output_direction(entry_dir: int) -> int:
    var rel_in  = (entry_dir - facing + 4) % 4
    var rel_out = -1
    match tile_type:
        TileType.STRAIGHT:
            if   rel_in == 2: rel_out = 0
            elif rel_in == 0: rel_out = 2
        TileType.CORNER_L:
            if   rel_in == 2: rel_out = 3
            elif rel_in == 1: rel_out = 0
        TileType.CORNER_R:
            if   rel_in == 2: rel_out = 1
            elif rel_in == 3: rel_out = 0
    if rel_out == -1: return -1
    return (rel_out + facing) % 4

func direction_to_delta(dir: int) -> Vector2i:
    match dir:
        0: return Vector2i(0, -1)   # North
        1: return Vector2i(1,  0)   # East
        2: return Vector2i(0,  1)   # South
        3: return Vector2i(-1, 0)  # West
    return Vector2i.ZERO

shader_type spatial;
render_mode blend_mix;
uniform vec4 ghost_color : source_color = vec4(0.0, 0.9, 1.0, 0.45);
uniform float pulse_speed : hint_range(0.5, 5.0) = 2.0;
void fragment() {

    ALBEDO = ghost_color.rgb;

}

shader_type spatial;
uniform sampler2D arrow_texture : source_color;
uniform float scroll_speed = 1.0;

    ALBEDO = texture(arrow_texture, uv).rgb;
}
